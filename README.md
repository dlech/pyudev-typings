pyudev-typings
==============

Type stubs for the [pyudev](https://github.com/pyudev/pyudev) library.

Usage
-----

Include this as a submodule in your project and point your type checker to it.

For example, Pylance/Pyright will look in `typings/` folders automatically.

    git submodule add https://github.com/dlech/pyudev-typings.git typings/pyudev
